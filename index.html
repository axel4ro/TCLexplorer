The Cursed Land - game

Free to Play Web2, Web3 and AI MMORPG cross-platform/p>

(Massively Multiplayer Online Role-Playing Game)

developed by PopuGames for mobile devices and PC.

Available for Android, iOS, Windows and soon MacOS.

Races

Human

  • Can use any Weapons
  • Can use any Class Skills
  • Can use only Human Armours

Beast

  • Can use any Weapons
  • Can use any Class Skills
  • Can use only Beast Suits

Undead

  • Can use any Weapons
  • Can use any Class Skills
  • Can use only Undead Plates

Elf

  • Can use any Weapons
  • Can use any Class Skills
  • Can use only Elf Clothings

Classes

Shaman

Assassin

Races: Human, Beast, Undead, Elf

Hunter

Can be used by any Race

Races: Human, Beast, Undead, Elf

Hunter Skills can be used with Bow weapon

Warrior (Soon)

Can be used by any Race

Races: Human, Beast, Undead, Elf

Wizard (Soon)

Can be used by any Race

Races: Human, Beast, Undead, Elf

Wizard Skills can be used with Staff weapon
